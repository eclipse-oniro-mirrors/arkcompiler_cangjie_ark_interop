# ohos.bluetooth.constant（蓝牙constant模块）

constant模块提供了蓝牙中常量的定义。

## 导入模块


## enum ProfileConnectionState

```cangjie
public enum ProfileConnectionState <: Equatable<ProfileConnectionState> & ToString {
    | StateDisconnected
    | StateConnecting
```

- ToString
