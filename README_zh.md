# 仓颉 API 公共仓

## 简介

仓颉 API 公共仓，包含仓颉互操作库, 公共的 json 库/ APILevel 库/ cffi 库等。

## 目录

```
.
├── api                   # 仓颉 api 的公共仓代码
├── mocks                 # sdk 构建时 ffi mock 桩
├── generate_mock.py      # 生成 mock 的构建脚本
├── bundle.json
└── LICENSE
```

## 使用场景
包含仓颉- ArkTS 互操作库, 用于仓颉- ArkTS 互操作. json 库/ APILevel / cffi 库为基础功能库, 提供给仓颉 api /仓颉 ui 等子系统等公共能力.
